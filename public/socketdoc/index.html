SMTP Map API 0.1.0

Apache 2.0

L'API SMTP Map permet de gérer la flotte de camion des chantiers

Fonctionnalités principales:

  • Envoyer / Recevoir en temps réel la position GPS des camions
  • Se connecter à différents chantiers

Servers
